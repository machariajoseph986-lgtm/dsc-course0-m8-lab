This repository contains an analysis of aviation accident data (1948–2023) aimed at identifying patterns in aircraft safety, including injury severity, aircraft damage, and operational risk factors. The analysis follows the CRISP-DM methodology.

## Repository Contents

* `CRISP_DM.txt`
  Project methodology and structured CRISP-DM breakdown.

* `Aviation_Accidents_Cleaning.ipynb`
  Data loading, cleaning, preprocessing, and feature engineering.

* `Aviation_Accidents_Data_Analysis.ipynb`
  Exploratory data analysis, visualizations, and insights.

* `aviation_cleaned_final.csv`
  Cleaned dataset after preprocessing.

* `aviation_engineered.csv`
  Dataset with engineered features used for analysis.

* `aviation_filtered_clean.csv`
  Filtered dataset used for modeling and EDA.
  
* `Aviation_Accident_Analysis_Report`
  Filtered dataset used for modeling and EDA.

## Disclaimer

This analysis is exploratory in nature and does not establish causal relationships or predictive models. Results are based on historical aviation accident data (1948–2023) and are intended for educational and analytical purposes only.
